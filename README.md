# claude-config (Deprecated)

> **This package has been renamed to [coder-config](https://github.com/regression-io/coder-config).**

## Migration

```bash
# Uninstall old package
npm uninstall -g @regression-io/claude-config

# Install new package
npm install -g coder-config
```

Your settings in `~/.claude-config/` are preserved automatically.

## New Repository

All development has moved to: **https://github.com/regression-io/coder-config**

- Issues: https://github.com/regression-io/coder-config/issues
- Releases: https://github.com/regression-io/coder-config/releases

## Why the rename?

`coder-config` now supports multiple AI coding tools:
- Claude Code
- Gemini CLI  
- Codex CLI
- Antigravity

The new name better reflects its multi-tool nature.
